Proyecto de Lenguaje de Interfaz

Con el Ing. Lagunas

Este proyecto tiene como objetivo crear una interfaz que permita controlar tres objetos físicos mediante una placa ESP32. La comunicación entre la interfaz y el microcontrolador se realizará a través de un protocolo adecuado (por ejemplo, serial, WiFi o Bluetooth).

Se desarrollará una interfaz visual amigable que facilite la interacción del usuario para activar o desactivar dispositivos conectados, como pueden ser: luces, motores o ventiladores.

Este trabajo integrador busca aplicar conocimientos de diseño de interfaz, programación web o de escritorio, y comunicación con hardware.
