Proyecto De Lenguajes De Interfaz

Este proyecto demuestra cómo encender un foco de 125V desde una página web utilizando componentes electrónicos básicos y un microcontrolador. El sistema permite el control remoto de un foco mediante una interfaz gráfica accesible desde cualquier navegador conectado a la misma red local.

Para lograrlo, se utiliza un relevador (relé), que actúa como un interruptor electromecánico. Este componente permite abrir o cerrar un circuito de corriente alterna (AC) de 125V, que es la tensión necesaria para encender un foco convencional. El relevador es activado por una señal de 3.3V o 5V proveniente del ESP32.

El transistor (generalmente tipo NPN como el 2N2222 o el TIP120) se utiliza como un puente entre el microcontrolador y el relevador. Dado que el ESP32 no puede entregar suficiente corriente directamente para energizar la bobina del relevador, el transistor amplifica la señal. Cuando recibe la señal desde el ESP32, permite que fluya corriente suficiente desde una fuente externa hacia el relevador, activándolo.

Para proteger el circuito, se incorpora un diodo de rueda libre (por ejemplo, 1N4007) en paralelo con la bobina del relevador. Esto es esencial porque cuando el relevador se desactiva, su bobina puede generar un pico de voltaje inverso (retroceso inductivo) que podría dañar el transistor o el microcontrolador. El diodo disipa este voltaje, protegiendo los componentes.

Todo el sistema está controlado mediante una página web alojada en el ESP32, diseñada con HTML, CSS y JavaScript. Desde esta página, el usuario puede encender o apagar el foco con solo hacer clic en un botón. El ESP32 actúa como servidor web, recibe la petición y envía la señal al transistor para activar el relevador.

Esta solución es un ejemplo práctico de automatización del hogar o control domótico. Su implementación demuestra la capacidad de integrar electrónica de potencia con interfaces web modernas, brindando una experiencia interactiva, educativa y funcional. También es una base ideal para expandir hacia proyectos más complejos como control por voz, integración con sensores o sistemas de seguridad.
